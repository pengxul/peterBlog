CKEditor Markdown
=================

Description
===========
The Markdown plugin provides the ability to use markdown to style the content
in your CKEditor field.

Usage
=====
Go to the Text formats and editors settings (/admin/config/content/formats) and
add the Markdown Button to any CKEditor-enabled text format you want.
